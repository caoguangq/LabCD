# LabCD

遥感变化检测标注工具，参考[EISeg](https://github.com/PaddlePaddle/PaddleSeg/tree/release/2.6/EISeg)，作为自己学习C++和Qt的练习项目。

## 环境

- Visual Studio 2019

- Qt 6.4.0
